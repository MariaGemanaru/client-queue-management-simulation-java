# client-queue-management-simulation-java
Java application for simulating client queues and service processing using multithreading, synchronization mechanisms, and a graphical user interface.
## Features
- Generate clients randomly based on simulation parameters
- Simulate multiple queues operating simultaneously
- Assign each client to the queue with the minimum waiting time
- Track queue evolution in real time
- Compute statistics such as:
  - average waiting time
  - average service time
  - peak hour of the system
- Export simulation events to log files
- Provide a graphical interface for configuring and visualizing the simulation
## Technologies Used
- Java
- Java Swing (Graphical User Interface)
- Multithreading
- Synchronization
- Object-Oriented Programming
